Class Tweet
In: lib/tweet.rb
Parent: Object

Object representing a single Tweet. Use the methods in TwitterScour to get Tweets.

Attributes

author_name  [RW]  Name of the author of this tweet, without the @ symbol. If the tweet was a straight retweet without any added text, this is the original author.
author_pic  [RW]  URL for the author’s picture/avatar
location  [RW]  A TweetLocation instance containing the location of the tweet, if the tweet has a location
text  [RW]  The full text of the tweet. This does not contain any HTML/links.
time  [RW]  Ruby Time for when this tweet was published
